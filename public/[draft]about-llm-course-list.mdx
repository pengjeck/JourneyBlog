# 目标************

## 初衷

因为看到推特上很多人分享一些大厂免费的LLM课程，获得了很多的关注很点赞，觉得很多人对这方面的信息是有需求的。想到，或许我可以做一个课程聚合网站把这些信息收集起来，方面大家检索学习。

## 期望

1. 通过这个网站给其他产品包括自己的社交账号引流，获得更多关注和流量
2. 如果有很多人搜索浏览相关内容，可以挂点广告，收点广告费。
3. 当做自己做内容导航网站的开始，毕竟这也是一个非常大的品类。

# 结果

> 对照期望，逐点检验

因为内容单调，而且因为选的是文档类的框架，更适合文档而不是资源分享站，最后网站没有发布

1. 网站没有发布，所以没有流量

3. 整理了包含Google，OpenAI，Amazon，Microsoft，Stanford和HuggingFace等公司的主要课程，包含课程地址，课程简介等。所有信息都放到了Google SpreadSheet上用公开链接的方式发布出来
4. 摸索半天，用[starlight](https://starlight.astro.build/)，写好了网站，把Google SpreadSheet上的数据展示出来，代码也开源到了[github上](https://github.com/pengjeck/CollectionList)
5. 在推特和Reddit上发了两个post，分享了Google SpreadSheet的链接。
