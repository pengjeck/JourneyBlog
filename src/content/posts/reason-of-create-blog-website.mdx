# 延伸的问题：为什么要搭建自己的博客网站？

为什么要搭建自己的博客网站，而不是是直接用medium，微信公众号，csdn这类博客平台？

首先，需要强调的是，搭建自己的博客网站并不代表着放弃medium这类平台，这不是一个二选一的问题。事实上，未来我也会考虑在medium，微信公众号上写作。

但我仍然觉得搭建个人博客网站是必要的，主要有两个原因：
1. 个人博客在管理上更加灵活，一些有趣的功能可以在自己的博客上实验，比如说SEO，友链，评论，newsletter。
2. 产品流量矩阵，简单来说，就是利用博客给产品导流，产品给博客导流。

# 延伸问题：选择合适的博客网站

个人观点是，我们作为创作者在互联网上创造的一切内容，所有权都是我们个人。但是总有一些道德下限极低的平台UGC平台，完全无视所有权的概念，将内容产生的收益几乎全部鲸吞，或者只把极少部分分给创作者。并且这些平台在运营的过程中，想尽一切办法最大化自己的收益，在商家和用户之间两头压榨。

在我以前认识不到这个问题的时候，在国内的博客平台上发表过123篇文章，有58.8w的访问。现在随便点开其中一篇博客，都能看到几个广告和一串相关文章推荐，而这些广告和推荐所产生的收益没有凭空消失，只是流到了平台的口袋。

所以在选择托管博客的时候，我会特别在意平台运营者对于内容和创作者的态度，以及平台的收益分配。
