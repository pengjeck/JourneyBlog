# 工具栈集合
https://seo.box/

集合了一些最常用的SEO工具，包括SimilarWeb，Semrush，Ahrefs等常用SEO工具站，还有一些优秀的SEO插件和SEO博客，如果不知道怎么入门，可以从这里开始进到不同的网站里面看看。

![seo box](/images/post/seo-box.png)

# 关键词分析
- [semrush](https://www.semrush.com/analytics/keywordoverview) 每天有十次免费机会

- [Ahrefs keyword generator](https://ahrefs.com/keyword-generator/) 根据输入的关键词，找到相关的query和匹配的短句

- [Ahrefs keyworkd difficulty ](https://ahrefs.com/keyword-difficulty) 分析某个关键词的困难程度，配有一些提醒
![seo keyword difficulty](/images/post/seo-keyword-difficulty.png)

# 帮助SEO优化的工具

[ahrefs site audit](https://app.ahrefs.com/site-audit)
- 可以手动触发网站爬虫，查看网站的SEO情况，爬的过程有点慢，我的博客网站249链接，用了16分钟才爬完
- 基于爬虫的结果给网站的SEO健康度打分，满分100
- 会把所有的问题列出来，包括常见的孤儿页面，https/http混合，404，308重定向等，照着issue去优化就好了
  - 问题有可能会比较多，但是问题是分Error，Warning和Notice三级的，挑重点的先优化

[ahrefs seo toolbar（浏览器插件）](https://ahrefs.com/seo-toolbar)
- 虽然ahrefs很强大，但是本地调试的时候用不了，如果你想快速的验证自己的修改效果，可以localhost打开之后用这个插件来检测一些常见问题
